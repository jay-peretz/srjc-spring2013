This class is designed to instruct you in how to create dynamic web applications using HTML5, CSS3, Javascript, JQuery, PHP and MySQL. While I will be using Adobe Dreamweaver in the class lectures and there are many features of Dreamweaver that help us as a development tool, we will be focusing on the technologies and techniques and so Dreamweaver is not necessarily required if you have a different preference (or limited budget) for development tools. This course is in transition and will focus on the server-side programming techniques using PHP, MySQL and techniques such as AJAX, JSON and REST.

This is a lecture based class in Petaluma on Wednesday evenings at 7PM. That said, attendance in class is not required and all lectures will be broadcast, captioned and archived for later viewing. While we encourage you to attend in person, it is possible to live out of town and take this class.
